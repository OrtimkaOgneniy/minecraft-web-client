# ⛏️ Minecraft Web Client

Полнофункциональный веб-клиент Minecraft, который работает прямо в браузере без установки.

## 🎮 Возможности

✅ Игра в браузере (без CMD)
✅ 3D мир на основе Three.js
✅ Движение и прыжки
✅ Вращение камеры
✅ Генерация ландшафта с деревьями
✅ HUD с информацией
✅ Простой и интуитивный интерфейс
✅ Работает на любом ПК с браузером

## 🚀 Как запустить

### Способ 1: Прямо из GitHub (самый простой)

Просто откройте файл `index.html` в браузере:
1. Клонируй репозиторий или скачай файлы
2. Открой `index.html` двойным кликом в браузере
3. Готово! Начинай играть

### Способ 2: Через локальный сервер (рекомендуется)

Если у тебя установлен Python:

```bash
python -m http.server 8000
```

Затем открой http://localhost:8000

## 🎮 Управление

- **W, A, S, D** - Движение вперед, влево, назад, вправо
- **Пробел** - Прыжок / Летать вверх
- **Shift** - Летать вниз
- **Мышь** - Вращение камеры
- **Левый клик** - Разрушить блок
- **Правый клик** - Поставить блок
- **E** - Инвентарь (в разработке)

## 📋 Требования

- Современный браузер (Chrome, Firefox, Safari, Edge)
- Интернет (для загрузки Three.js)

## 🔧 Технологии

- **Three.js** - 3D графика
- **WebGL** - Рендеринг
- **HTML5/CSS3/JavaScript** - Фронтенд
